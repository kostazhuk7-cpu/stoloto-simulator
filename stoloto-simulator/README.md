# Столото Симулятор v2

**Симулятор 15 игр Спортлото с параллельным запуском, стратегиями, реальными архивами и сравнением стратегий в реальном времени.**

## Что это

Веб-симулятор для анализа лотерей Столото. Каждый тираж играется сразу ДВУМЯ стратегиями — выбранной пользователем и фоновой «по генератору» — с отдельными колонками в таблице для сравнения.

## Возможности

- **15 игр**: 6 из 45, 5 из 36, 7 из 49, 4 из 20, 4 из 20 Мини, 6 из 45 Плюс, Большое Спортлото, Рапидо, Рапидо Про, Рапидо Старт, Охота, Охота. Вызов, КЕНО, Проще чем дважды два, Забава + Русское Лото, Жилищная (бинго)
- **4 ГСЧ**: Math.random, Crypto, Time-seed, Hybrid + случайный выбор каждый тираж
- **5 стратегий**: Случайный, Горячие числа (история), Холодные числа, Смешанная, и фоновая «По генератору»
- **Сравнение Gen**: Каждый тираж параллельно играет стратегию «по генератору» — результат в отдельных колонках (Gen) в сводной таблице
- **Реальные тиражи**: 200+ тиражей по каждой игре (собраны со stoloto.ru), отображаются в виде прогноза горячих чисел
- **Логи генератора**: По каждому полю отдельно отслеживается частота выпадающих чисел от ГСЧ — отображается в карточках игр
- **Бинго-конкуренция**: Реалистичное моделирование с 500+ конкурентами
- **Валидация**: Проверка корректности правил каждой игры
- **Экспорт**: CSV и JSON

## Запуск

```bash
# Через run.bat (рекомендуется — локальный сервер)
run.bat

# Или вручную
cd stoloto-simulator
python -m http.server 8080
# Открыть http://localhost:8080
```

Локальный сервер обязателен — браузер блокирует загрузку JSON-файлов через file://.

## Использование

1. Выбери игры (галочки на карточках)
2. Настрой ГСЧ и стратегию
3. Нажми **▶ Запустить**
4. Смотри сводную таблицу внизу — колонки Gen показывают результат стратегии «по генератору»
5. Нажми **Загрузить и сравнить** — прогноз на основе реальных архивов stoloto.ru

## Обновление архивов

```bash
cd stoloto-simulator
python fetch_real_draws.py
# Собирает свежие тиражи для всех 15 игр с stoloto.ru
```

## Стратегии

| Стратегия | Описание |
|-----------|----------|
| **Случайный** | Случайный выбор каждый тираж (контроль) |
| **По генератору (фон)** | Отслеживает частоты ГСЧ по каждому полю, подбирает самые частые числа. Работает всегда параллельно с основой стратегией |
| **Горячие числа** | Выбирает числа, которые чаще всего выпадали в истории симуляции |
| **Холодные числа** | Выбирает числа, которые реже всего выпадали |
| **Смешанная** | Комбинация горячих и случайных |

## Ключевые инсайты

- Матожидание **всегда отрицательное** — лотерея не способ заработать
- Стратегия «по генератору» показывает те же результаты, что и случайная — что математически верно
- Разные игры имеют разный ROI — сводная таблица показывает лучшие/худшие
- Конкуренция в бинго сильно снижает реальные шансы
- Частотный анализ реальных тиражей stoloto.ru подтверждает равномерное распределение
