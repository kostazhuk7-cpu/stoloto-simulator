# 🎰 Столото Симулятор v2

**Симулятор всех игр Спортлото с параллельным запуском, мультивыбором и сравнением шансов.**

## Что это

Веб-симулятор для анализа лотерей Столото. Позволяет запускать все игры параллельно с разными стратегиями и ГСЧ, чтобы понять реальные шансы на выигрыш.

## Возможности

- **12 игр**: 6 из 45, 5 из 36, 7 из 49, 4 из 20, Большое Спортлото, Всё или Ничего, Рапидо, Охота, КЕНО, Русское Лото, Жилищная лотерея, Зодиак
- **4 ГСЧ**: Math.random, Crypto, Time-seed, Hybrid + случайный выбор каждый тираж
- **4 стратегии**: Случайный, Горячие числа, Холодные числа, Смешанная
- **Мультивыбор**: Развёрнутая ставка для увеличения шансов
- **Бинго-конкуренция**: Реалистичное моделирование с 500+ конкурентами
- **Сводная таблица**: Сравнение ROI, шансов, суммы выигрышей по всем играм
- **Валидация**: Проверка корректности правил каждой игры
- **Логирование**: Полный JSON-экспорт для аналитики
- **Экспорт**: CSV и JSON

## Запуск

```bash
# Локально
python -m http.server 8080
# Открыть http://localhost:8080
```

Или просто открой `index.html` в браузере.

## Использование

1. Выбери игры (галочки на карточках)
2. Настрой ГСЧ и стратегию
3. Нажми **▶ Запустить** — симуляция работает пока не остановишь
4. Смотри статистику внизу: таблица с ROI, шансами, балансом

## Ключевые инсайты

- Матожидание **всегда отрицательное** — лотерея не способ заработать
- Разные игры имеют разный ROI — сводная таблица показывает лучшие/худшие
- Конкуренция в бинго сильно снижает реальные шансы
- Случайный ГСЧ каждый тираж не меняет матожидание

## Структура

- `index.html` — UI с тремя панелями
- `style.css` — тёмная тема, адаптивная сетка
- `script.js` — движок симуляции, 12 игр, валидация, логирование

## Технологии

- HTML5 / CSS3 / JavaScript (ES6+)
- Без зависимостей — работает в любом браузере
- Криптографический ГСЧ через Web Crypto API
